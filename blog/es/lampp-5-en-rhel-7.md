# LAMPP 5 en RHEL 7

Despliega un stack LAMPP con PHP 5.6 ajustado para Red Hat Enterprise Linux 7.

[toc]

## Requisitos

- VM RHEL 7 con acceso sudo
- Puertos HTTP/HTTPS abiertos

## Pasos generales

1. Instalar Apache y PHP 5.6
2. Configurar MySQL/MariaDB
3. Ajustar PHP y módulos
4. Probar con una página info.php

## Recursos

- Guía Apache: https://httpd.apache.org/docs/
- PHP 5.6: https://www.php.net/releases/5_6_0.php
