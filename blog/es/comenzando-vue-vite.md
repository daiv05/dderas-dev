# Comenzando con Vue 3 y Vite

Vue 3 combinado con Vite proporciona una experiencia de desarrollo increíblemente rápida. Esta guía te mostrará cómo configurar tu primer proyecto.

## Prerequisitos

Antes de comenzar, asegúrate de tener:

- Node.js 16+ instalado
- Conocimientos básicos de JavaScript
- Un editor de código (VS Code recomendado)

## Instalación

Crea un nuevo proyecto usando la herramienta oficial de Vite:

```bash
npm create vite@latest mi-app-vue -- --template vue
cd mi-app-vue
npm install
npm run dev
```

## Estructura del Proyecto

La estructura predeterminada incluye:

- `src/` - Código fuente de tu aplicación
- `public/` - Recursos estáticos
- `index.html` - Punto de entrada
- `vite.config.js` - Configuración de Vite
