# Cuenta .edu en Azure

Activa una cuenta de Microsoft Azure usando tu correo .edu y obtén crédito sin necesidad de tarjeta.

[toc]

## Requisitos

- Correo institucional .edu activo
- Acceso a la bandeja de entrada para verificación

## Pasos generales

1. Ir a la página de Azure para estudiantes
2. Registrarte con tu correo .edu
3. Verificar identidad y completar perfil
4. Reclamar el crédito promocional

## Recursos

- Azure for Students: https://azure.microsoft.com/free/students/
