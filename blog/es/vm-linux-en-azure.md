# VM Linux en Azure

Esta guía muestra cómo crear una máquina virtual Linux en Microsoft Azure desde cero.

[toc]

## Requisitos

- Cuenta en Azure activa
- Permisos para crear recursos

## Pasos generales

1. Crear grupo de recursos
2. Crear la VM con imagen Linux
3. Configurar red y puerto SSH (22)
4. Conectarse a la VM y validar
