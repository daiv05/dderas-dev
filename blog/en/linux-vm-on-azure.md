# Linux VM on Azure

This guide walks through creating a Linux virtual machine on Microsoft Azure from scratch.

[toc]

## Requirements

- Active Azure account
- Permissions to create resources

## Steps

1. Create a resource group
2. Create the VM with a Linux image
3. Configure networking and SSH port (22)
4. Connect to the VM and validate
