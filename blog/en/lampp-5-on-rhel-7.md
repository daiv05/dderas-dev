# LAMPP 5 on RHEL 7

Deploy a LAMPP stack with PHP 5.6 tuned for Red Hat Enterprise Linux 7.

[toc]

## Requirements

- RHEL 7 VM with sudo access
- Open HTTP/HTTPS ports

## Steps

1. Install Apache and PHP 5.6
2. Configure MySQL/MariaDB
3. Adjust PHP and modules
4. Test with an info.php page
